Utility Function to print 8x8 grid

# Generating 8x8 grid

def grid(idx):
    fig, ax = plt.subplots(8,8)
    fig.set_size_inches(16,9)
    fig.tight_layout()
    for i in range(0,8):
        for j in range(0,8):
            img = np.reshape(np.array((df.iloc[idx])[1:]), (28,28))
            ax[i][j].imshow(img, cmap='grey')
            ax[i][j].axis('off')
            ax[i][j].set_title(df.iloc[idx][0])
            idx += 1

Logistic Regression¶

Choice of solvers¶

Liblinear The solver uses a Coordinate Descent (CD) algorithm that solves optimization problems by successively performing approximate minimization along coordinate directions or coordinate hyperplanes.

Saga The SAGA solver is a variant of SAG that also supports the non-smooth penalty L1 option (i.e. L1 Regularization). This is therefore the solver of choice for sparse multinomial logistic regression. It also has a better theoretical convergence compared to SAG.
